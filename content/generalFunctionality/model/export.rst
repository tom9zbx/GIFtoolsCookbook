Export a model
==============

GIFtools will allow the user to export any model object at any time. 

Select the GIFmodel object then the menu **GIFmodel** |rarr| **Export** 


.. figure:: ../../../images/modelGen.png
    :align: center
    :width: 400


**NOTE** Exporting of the model will *not* export its associated mesh (see :ref:`Exporting meshes <exportMesh>`)!
