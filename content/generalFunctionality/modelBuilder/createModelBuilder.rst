Create a modelBuilder module
============================

The modelBuilder module allows the user to create reference models, weights, and bounds constraints for inversion from geology and other physical property measurements. To create a modelBuilder module, an *active cell model* (:ref:`ACTIVEmodel <modelfile>`) is required. See :ref:`here <createActiveCells>` on how to create an active cell model. Click on the active cell model and choose



**Model Builder** |rarr| **Start modelBuilder module**
